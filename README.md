# Income Spending Survey Tool

## Overview
This project collects and analyzes income and spending data to support a healthcare product launch.

## Technologies Used
- Flask (Web App)
- MongoDB (Database)
- Python (Data Processing)
- Pandas & Matplotlib (Visualization)
- AWS EC2 (Deployment)

## How to Run
1. Install dependencies
2. Start MongoDB
3. Run Flask app:
   python app.py
4. Submit data via browser
5. Analyze data in Jupyter Notebook

## Outputs
- CSV dataset
- Income vs Age chart
- Gender spending distribution chart
